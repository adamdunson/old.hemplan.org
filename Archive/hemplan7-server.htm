Downloads

  • There will be a shared folder on the network called HempLAN 7.0 Installs and HempLAN FTP for any files needed.
  • If you cannot access the network shared folder, you can attempt to access the FTP folder:
    • Open IE6 or Windows Explorer
    • Go to: ftp://192.168.1.100/
    • Username: hemplan_guest; Password: leetpeen
    • If that doesn't work, you must disable Passive Mode.
      1. Go to Tools>Internet Options and click the Advanced Tab
      2. Scroll down and deselect the option that says Use Passive FTP Mode (or something similar to that)
      3. Hit apply and close then reopen IE or Windows Explorer (whichever you are using).
      4. Now, attempt to connect to the FTP again with the same credentials.
